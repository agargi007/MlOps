# MLOps Project – Diabetes Prediction API

This project demonstrates a complete MLOps pipeline, including model training, API deployment, containerization, and CI/CD integration.

---

## 📌 Project Overview

This project builds a Machine Learning model to predict whether a person is diabetic based on health parameters. The model is served using FastAPI and deployed using Docker.

---

## 🚀 Features

- Train ML model and save as `.pkl`
- FastAPI backend for inference
- API testing using Swagger & Postman
- Docker containerization
- CI/CD pipeline using GitHub Actions
- Ready for cloud deployment
